Brands Template Plugin - Help Guide

Plugin Information

Plugin Name: Brands Template

Version: 1.1

Author: Choctaw Websites

Overview

The Brands Template plugin displays reviews related to a specific brand using the shortcode [reviews_by_brand]. The output matches the styling of the archive-template plugin, displaying reviews in a single-column layout with featured images, titles, rating images, and excerpts.

Files Included

  • brands_template.php - Main plugin file that registers and loads the plugin.
  • sc_reviews_by_brand.php - The shortcode file that handles queries and displays related reviews.
  • reviews_by_brand.css - The CSS file that styles the layout and visual elements.
  • help.html - This help document.

Shortcode Usage: [reviews_by_brand]

The [reviews_by_brand] shortcode is intended to be used in the "Brand Template" Element of GeneratePress.

How It Works

  1. When the shortcode is placed inside a "brands" post, it retrieves the Post ID of the current brand.
  2. It then queries all posts from the "reviews" custom post type where the "reviews_brand" field (ACF Relationship Field) matches the current brand.
  3. If matching reviews are found:
    • Each review is displayed in a single-column layout.
    • The featured image (120px width) appears on the left.
    • The review content appears on the right, including:
      • Post title (linked to the review)
      • Rating image (if available)
      • Post excerpt
  4. If no reviews are found, a message stating "No reviews available for this brand." is displayed.

CSS Styling and Shared Classes

Important Note: This plugin shares CSS class names with the archive-template plugin. Both plugins use identical class names such as:

  • .taxonomy-archive-posts - Main container for posts
  • .archive-post - Individual post container
  • .archive-post-image - Featured image container
  • .archive-post-title - Post title styling
  • .rating-on-archive - Rating image styling
  • .excerpt - Post excerpt styling

CSS Interaction: Due to the shared class names, any changes made to the CSS in either plugin will affect both displays. For example, modifying the margin for .rating-on-archive in this plugin will also change the spacing in the archive-template plugin.

Current Styling Details

  • Single-column layout with 120px image column and flexible content column
  • Links are colored #955733 with underline, changing to #222222 on hover
  • Compact spacing between title, rating, and excerpt
  • Responsive design that switches to vertical layout on mobile devices

Installation

  1. Upload & Activate:
    • Upload the "brands-template" folder to /wp-content/plugins/.
    • Activate the "Brands Template" plugin from the WordPress admin panel.
  2. Use the Shortcode:
    • Place [reviews_by_brand] inside any single "brands" post.

Troubleshooting

Q: The shortcode is not displaying anything.

  • Ensure you are using it inside a "brands" post.
  • Check if the brand has any associated "reviews".
  • Confirm that the "reviews_brand" field in ACF is correctly set up as a Relationship Field.

Q: Images are missing or not displaying correctly.

  • Ensure the reviews have featured images assigned.
  • If no featured image exists, a placeholder image will be displayed automatically.

Q: CSS changes are affecting both plugins.

  • This is expected behavior due to shared class names between brands-template and archive-template plugins.
  • If you need different styling for each plugin, contact the developer to implement unique class names.
