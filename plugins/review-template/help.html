Review Template Plugin - Help

Version: 1.0

Description

This plugin adds custom shortcodes for displaying review information on the Review Template element.

Included Files

  • review_template.php
  • sc_review_brand_name.php - Shortcode: [review_brand_name]
  • sc_review_establishment_number.php - Shortcode: [review_establishment_number]
  • sc_review_meat_type.php - Shortcode: [review_meat_type]
  • sc_review_variety.php - Shortcode: [review_variety]
  • sc_review_ingredients.php - Shortcode: [review_ingredients]
  • sc_review_allergens.php - Shortcode: [review_allergens]
  • sc_review_advertised_claims.php - Shortcode: [review_advertised_claims]
  • sc_review_snack_form.php - Shortcode: [review_snack_form]

Shortcodes & Usage

[review_brand_name]

Retrieves and displays the brand name associated with a review post, linking it to the brand's page.

[review_brand_name]

Example Output:

<a href="https://yoursite.com/brands/example-brand/">Example Brand</a>

[review_establishment_number]

Retrieves and displays the establishment number linked to a review.

[review_establishment_number]

Example Output:

<a href="https://yoursite.com/est/smith-meat-co/">Est# Smith Meat Co.</a>, (Smith Processing Facility)

[review_meat_type]

Displays the meat type(s) related to a review, each linked to its archive page.

[review_meat_type]

Example Output:

<a href="https://yoursite.com/meat-type/beef/">Beef</a>, 
<a href="https://yoursite.com/meat-type/pork/">Pork</a>

[review_variety]

Displays the variety type(s) related to a review.

[review_variety]

[review_ingredients]

Displays the ingredient(s) associated with a review post.

[review_ingredients]

[review_allergens]

Displays the allergen(s) related to a review.

[review_allergens]

[review_advertised_claims]

Displays the advertised claims linked to a review post.

[review_advertised_claims]

[review_snack_form]

Displays the snack form(s) related to a review, each linked to its archive page.

[review_snack_form]

Example Output:

<a href="https://yoursite.com/snack-form/sticks/">Sticks</a>, 
<a href="https://yoursite.com/snack-form/strips/">Strips</a>, 
<a href="https://yoursite.com/snack-form/chunks/">Chunks</a>
