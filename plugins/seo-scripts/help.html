SEO Scripts Plugin - Documentation

Plugin Information

Plugin Name: SEO Scripts

Version: 1.0

Author: Choctaw Websites

Description: Enhances SEO metadata for taxonomy archive pages used in the "reviews" custom post type.

Functionality Overview

This plugin improves the SEO of taxonomy archive pages by dynamically generating:

  • Custom <title> tags based on user-defined SEO fields.
  • <meta> description tags using either the taxonomy description or a fallback template.
  • Open Graph meta tags (og:title, og:description, og:image) for better social media previews.
  • Twitter Card meta tags (twitter:title, twitter:description, twitter:image) for optimized Twitter sharing.
  • JSON-LD Schema (CollectionPage) for better Google search visibility.

It works for the following taxonomies:

  • Meat Type (meat-type)
  • Ingredient (ingredient)
  • Claim (claim)
  • Variety (variety)
  • Allergen (allergen)
  • Snack Form (snack-form)

How It Works

  1. The plugin automatically detects when a taxonomy archive page is loaded.
  2. It retrieves data from Advanced Custom Fields (ACF) associated with the taxonomy term.
  3. If the required data exists, it generates <title>, <meta>, Open Graph, and Twitter Card tags.
  4. If no data is found in the fields, it falls back to default values.
  5. The generated meta tags are outputted in the <head> section of the HTML.

Required Data - Advanced Custom Fields

SEO Title (Used for <title>, Open Graph, and Twitter Cards)

  • Meat Type: meat-type-seo-title
  • Ingredient: ingredient_seo_title
  • Claim: claim_seo_title
  • Variety: variety_seo_title
  • Allergen: allergen_seo_title
  • Snack Form: snack_form_seo_title

Taxonomy Description (Used for <meta> description)

This is the built-in WordPress taxonomy description. If empty, a default description is generated based on the taxonomy name.

Taxonomy Image (Used for Open Graph & Twitter Cards)

  • Meat Type: meat-type-image
  • Ingredient: ingredient_image
  • Claim: claim_image
  • Variety: variety_image
  • Allergen: allergen_image
  • Snack Form: snack_form_image

Testing & Verification

Check Page Source

  • Open a taxonomy archive page.
  • Right-click -> "View Page Source".
  • Look for:
    • <title> tag
    • <meta name="description"> tag
    • Open Graph tags
    • Twitter Card tags

Validate Schema Markup

Use Google Rich Results Test to check for CollectionPage Schema.
