Fixed Header Plugin Help

Plugin Information

Plugin: Fixed Header

Description: Keeps the full-sized header fixed at the top of the page while scrolling.

Version: 1.1

Included Files

  • /plugins/fixed-header/fixed-header.php - The main plugin file
  • /plugins/fixed-header/fixed-header.js - JavaScript to adjust page content margin when header height changes
  • /plugins/fixed-header/fixed-header.css - CSS to style the fixed header and navigation

Functionality

  • Fixed Header: The .site-header class is styled to remain fixed at the top of the page.
  • Navigation Placement: The .main-navigation class must be positioned directly below .site-header in the document structure.
  • Admin Bar Handling: The plugin automatically detects and adjusts for the WordPress Admin Bar if present.
  • Content Adjustment: The JavaScript function adjustMargin() dynamically sets the .site-content margin based on the combined height of .site-header, .main-navigation, and the WordPress Admin Bar (if applicable).
  • Responsive Behavior: The plugin ensures the correct margin is applied when the window is resized.

Installation & Activation

  1. Upload the fixed-header plugin folder to /wp-content/plugins/.
  2. Activate the plugin in WordPress Admin â†’ Plugins.
  3. Ensure that .main-navigation is positioned below .site-header in your theme's HTML structure.

Notes

  • This plugin is designed for themes where the navigation is separate from the header. If your navigation is inside the header, additional styling may be required.
  • If the header overlaps navigation or content, adjust styles in fixed-header.css or modify the adjustMargin() function in fixed-header.js.
